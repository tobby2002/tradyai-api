A python module specialized on genetic algorithms using various representations. 
Intended to evolve a dict of parameters, nested or not, provided with respective ranges.

```
sampleParameters = {'short': (6,8),
              'persist': (1,50),
              'variableY': (2,6),
              'ROP_weight': (5,7),
              'santa': (1,10),
              'thresholds': {
                 'top': (8,11),
                 'bottom': (17,32)
              },
              'IL12': (3,8)}
```
